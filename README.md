# execute
编译从语言文件并执行

## 执行环境
- 仅限于Linux,建议用于装有gnome的系统

## 安装卸载

1. **安装**
- 先用终端移动到本目录内
- 然后再在终端中用管理员用户执行`./install.sh`命令
- *install.sh*文件会自动安装

2. **卸载**
- 先用终端移动到本目录内
- 然后再在终端中用管理员用户执行`./uninstall.sh`命令
- *uninstall.sh*文件会自动卸载

## 使用
- 在安装完成后会在桌面和应用程序栏创建快捷方式（启动器）
- 桌面的需要手动激活，右键*执行.desktop*文件，点击*允许启动*即可激活完成
- 可以在终端输入`execute`来执行程序
- 点击快捷方式
